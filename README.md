### 사용된 도구
* Spring-boot - Spring-boot 웹 프레임워크
* Maven - 의존성 관리 프로그램
* Tomcat - 웹 애플리케이션 서버
* IntelliJ IDEA - IDE
* MySql - RDMS
* Workbench - Mysql 툴
* AWS EC2 - 클라우드 환경 컴퓨팅 시스템
* AWS RDS - 클라우드 환경 데이터베이스 관리 시스템
* AWS S3 - 클라우드 환경 객체 스토리지
